# CineTune
CineTune adalah sebuah game tebak film interaktif yang menggabungkan elemen multimedia seperti audio, gambar, dan interaksi kamera. Pemain ditantang untuk menebak judul film dari gambar yang ditampilkan, dan uniknya, mereka harus menjawab menggunakan gesture tangan yang dideteksi secara real-time oleh kamera.
